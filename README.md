# 📊 Loan Approval Prediction using Python & ML Algorithms

### ✅ Project Details:

- Client – York City Bank, Vermont, USA
- Segment – Banking Domain
  
In this project, I analysed York City Bank’s Loan data for the year 2025. Used Machine Learning algorithms to develop a Training (80%) and Test (20%) dataset, performed 5-fold cross-validation, and performed Hyperparameter Tuning for better accuracy. 

### ⚒️ Tools used:

-	Jupyter Notebook
-	Python Library (NumPy, Pandas) 
-	Machine Learning Models
  
### ✅ End-to-End Process of the project: 

1. SOW - Understanding the ‘Scope of Work’ and performing tests accordingly to provide better insight generation
2. Data Collections - Collected the required data from the Organisation's Databases in the form of CSV files. Imported those CSV files to Jupyter Notebook for further transformation to make it ready for analysis.
3. Data Preparation & EDA – Cleaning data, removing null or missing values & handling categorical columns
4. Performed ‘Feature Scaling’, which is essential for ML Algorithms that normalize the range of independent variables.
   
-	Some of the ‘Feature Scaling’ techniques:
  
➡️ K-neighbours

➡️ Support vectors

➡️ Neural Network

➡️ Logistic Regression, etc.

 🔎 Conclusion:
 
After performing hyperparameter tuning, it was found that ‘Random Forest Classifier’ (RFC) had the best improvement in accuracy from 77.76 to 80.66%. So, selected 'Random Forest Classifier' model for the entire dataset/production.
